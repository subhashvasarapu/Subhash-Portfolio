# Personal Portfolio — Dark Theme

A single-file portfolio site (pure HTML/CSS, no build step) ready for GitHub Pages.

## Deploy (same as the roadmap site)
1. Create a new repo named exactly: `<your-username>.github.io`  ← this makes it your ROOT site (https://your-username.github.io)
   — or any other name (e.g. `portfolio`) for https://your-username.github.io/portfolio/
2. Upload `index.html` (Add file → Upload files → Commit)
3. Settings → Pages → Deploy from branch → `main` / `(root)` → Save
4. Live in ~2 minutes

## Before you publish — find & replace these placeholders in index.html:
| Placeholder | Replace with |
|---|---|
| `Your Name` (2 places) | your actual name |
| `your-name` (nav logo) | your name/handle |
| `YOUR-USERNAME` (all links) | your GitHub username |
| `YOUR-LINKEDIN` | your LinkedIn slug |
| `you@example.com` | your email |
| `https://YOUR-USERNAME.github.io/ai-transition-roadmap/` | your live roadmap URL |

## Updating over time
- Project cards: flip badges by changing the class — `badge plan` → `badge wip` → `badge ship` (Planned → In Progress → Shipped); update the badge text too
- Writing section: swap "coming soon" links for real blog post URLs as you publish
- Everything is one file — edit index.html directly on GitHub if you like
